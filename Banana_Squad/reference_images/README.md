# Reference Images

This folder is for "style inspiration" references (layout ideas, typography treatments, visual hierarchy patterns).

## VisualCapitalist Collector

If you run the collector script, it stores:
- images under: `Banana_Squad/reference_images/style_inspiration/visualcapitalist/downloads/`
- metadata under: `Banana_Squad/reference_images/style_inspiration/visualcapitalist/sources.json`

Notes:
- Collector is capped (default <= 50) and rate-limited.
- Keep attribution metadata. Do not remove URLs from `sources.json`.
