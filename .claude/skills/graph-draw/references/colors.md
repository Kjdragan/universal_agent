# Excalidraw Color Reference

Use colors to make diagrams visually clear and professional.

---

## Primary Shape Colors

Use these for main diagram elements:

| Color | Background | Stroke | Use For |
|-------|------------|--------|---------|
| Blue | `#a5d8ff` | `#1971c2` | Primary steps, main flow |
| Green | `#b2f2bb` | `#2f9e44` | Success, completed, positive |
| Orange | `#ffd8a8` | `#e8590c` | Highlights, important items |
| Purple | `#d0bfff` | `#7950f2` | Special items, unique elements |
| Yellow | `#fff3bf` | `#fab005` | Warnings, decisions, notes |
| Red | `#ffc9c9` | `#e03131` | Errors, problems, blockers |
| Cyan | `#b3e5fc` | `#0288d1` | Info, neutral steps |
| Gray | `#e9ecef` | `#868e96` | Disabled, future, neutral |

## Best Practices

1. **Use consistent colors** - Same color for same type of element
2. **Light backgrounds, dark strokes** - Ensures readability
3. **Limit palette** - 3-4 colors per diagram max
4. **White text on dark headers** - For swimlane headers
5. **Gray for secondary elements** - Arrows, borders, disabled items
